# Student Performance Analysis Report

## Objective
To analyze student academic performance using subject-wise marks.

## Analysis Summary
- Total and average marks were calculated for each student.
- Class average was computed.
- Top-performing student was identified.

## Insights
- Science has the highest average score.
- Some students consistently perform well across subjects.
- Overall class performance is good.

## Conclusion
The analysis helps understand student strengths and areas for improvement using simple data analysis techniques.
