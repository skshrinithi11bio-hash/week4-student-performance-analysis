# Week 4 – Student Performance Analysis

##  Project Overview
This project is part of the **Week 4 internship task – Complete Data Analysis Project**.  
It focuses on analyzing student academic performance using Python by calculating total and average marks, identifying the top-performing student, and visualizing performance trends.

The dataset is created and stored in Google Drive and accessed through Google Colab for analysis.

---

##  Objectives
- Analyze student marks across multiple subjects
- Calculate total and average marks for each student
- Identify the top-performing student
- Visualize student-wise and subject-wise performance
- Derive meaningful insights from academic data

---

## Tools & Technologies
- Python  
- Google Colab  
- Pandas  
- Matplotlib
